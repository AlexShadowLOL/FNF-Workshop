# How to Add Credits on Psych Engine

### 1. Method

```haxe
['Category Name'],
['Team Member Name', 'icon image name', 'Role/Description', '1 social media link', '444444'], // hex color
[''], // this separates each category if u put it above the og credits, add this if u put it at the whole bottom, delete this
```

**EX:**

```haxe
['VS YoMom'],
['AlexShadow', 'alexshadow', 'Programmer of The Mod', 'https://twitter.com/AlexDaShadow', 'C30085'], // self promo :troll:
['IdkDontCare', 'idk', 'Sat There.. Staring', 'https://twitter.com/YourAssSmellsGood', 'FF9300'], // if u wanna add another member
[''],
```
